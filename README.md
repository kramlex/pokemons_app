# MobX Timer

Um pequeno projeto construindo um Timer para demonstrar o uso da gerência de estado com o MobX

- Version: Flutter 3.3.2

### Run
1. Clone o repositório
2. Execute `flutter pub get`
3. Execute `flutter run`
